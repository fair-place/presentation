# layout-fix

Playwright でスクリーンショットを撮影し、視覚的にレイアウト崩れを検出・修正するスキル。

## 前提条件
- Playwright がインストールされていること
- ビルド済みの HTML が dist/ に存在すること

## 手順

1. プレビューサーバーを起動する
   ```bash
   bash scripts/start-server.sh
   ```

2. スクリーンショットを撮影する
   ```bash
   bash scripts/take-screenshots.sh
   ```

3. 撮影したスクリーンショットを確認し、以下の観点でレイアウト崩れを検出する
   - テキストがスライド領域からはみ出していないか
   - グリッドレイアウトが意図通りに配置されているか
   - パネルの角丸・影・ボーダーが正しく表示されているか
   - フォントサイズが適切か（小さすぎないか）
   - 余白が均等か
   - 背景画像が正しく表示されているか

4. 崩れが見つかった場合、対象スライドの Markdown を修正する
   - Tailwind CSS クラスの調整
   - テキスト量の削減
   - レイアウトパターンの変更（slides/example.md を参照）

5. 修正後、再度ビルド → スクリーンショット → 確認のサイクルを繰り返す

6. 完了後、クリーンアップする
   ```bash
   bash scripts/cleanup.sh
   ```
