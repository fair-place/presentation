# スタイルガイド

## デザイン哲学

1. **信頼・誠実** - 数字を隠さない、約束を守る
2. **透明・クリーン** - 明快なコミュニケーション
3. **親しみやすさ** - 堅苦しくない、話しかけやすい
4. **可読性重視** - 十分なフォントサイズとコントラストを確保する
5. **1スライド1メッセージ** - 情報過多を避ける

## 文体ルール

### 基本スタンス

- **丁寧だが堅苦しくない** - 「です・ます」調、業界用語を避ける
- **数字と事実で話す** - 「業界最高水準」より「マージン○%を開示」
- **対等な関係** - 上から目線NG、下手に出すぎもNG

### 表記ルール

- コロン（：）を使わない
  - NG: 「原則：自律性の最大化」
  - OK: 「原則 - 自律性の最大化」「原則❶ 自律性の最大化」
- 感嘆符（!）や疑問符（?）を避ける
  - NG: 「生産性が10倍に!」
  - OK: 「生産性が10倍に」
- 装飾的な絵文字を避ける
  - NG: 🎉🚀✨ など雰囲気だけの絵文字
  - OK: 📊🔄🎯 など意味を持つアイコン的使用は許容

### 人称

- 自社 - 「フェアプレイス」「私たち」（「弊社」は書類のみ）
- 相手 - 「お客様」「パートナー」
