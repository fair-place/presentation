# スタイルガイド

## デザイン哲学

1. **美麗** - 洗練されたビジュアルで聴衆の関心を引く
2. **シンプル** - 1スライド1メッセージ、情報過多を避ける
3. **色数制限** - グレースケール基調、アクセントは1スライドあたり1-2色まで
4. **可読性重視** - 十分なフォントサイズとコントラストを確保する

## カラーパレット

### プライマリグラデーション

```
linear-gradient(to right, #1B4565, #3E9BA4)
用途: タイトルスライドの背景、セクション区切り
```

### グレートーン（基本配色）

| トークン | HEX | 用途 |
|---------|-----|------|
| gray-50 | #f9fafb | パネル背景（明）、左カラム背景 |
| gray-100 | #f3f4f6 | パネル背景（標準）、右カラム背景 |
| gray-300 | #d1d5db | ボーダー、区切り線 |
| gray-400 | #9ca3af | 左パネルのボーダーアクセント（border-l-4） |
| gray-500 | #6b7280 | 右パネルのボーダーアクセント（border-l-4） |
| gray-600 | #4b5563 | サブテキスト |
| gray-700 | #374151 | 本文テキスト |
| gray-800 | #1f2937 | 見出しテキスト |

### 見出し特殊カラー

- `--h1-color: #246` — h1要素のデフォルト色
- `--heading-strong-color: #48c` — 見出し内の強調テキスト

### アクセントカラー（使用を限定）

- blue: `#0969da` / `#4493f8` — リンク、重要な強調
- red-600, green-600 は使用しない。gray系で代替する

## タイポグラフィ体系

### カスタムフォントサイズクラス（em単位）

| クラス | サイズ | 用途 |
|-------|-------|------|
| text-em-3xl | 3em | 数値強調、インパクトキーワード |
| text-em-2xl | 2em | パネル見出し、カード見出し |
| text-em-xl | 1.5em | セクション見出し、サブタイトル |
| text-em-lg | 1.25em | 本文テキスト、リスト項目 |
| text-em-base | 1em | 補足説明、キャプション |

- ルートフォントサイズ: 29px
- スライドサイズ: 1280 x 720px (16:9)
- パディング: 78.5px

## コンポーネント共通スタイル

### パネル/カード

```
角丸: rounded-xl
影: shadow-lg
パディング: p-6
ボーダー（左アクセント）: border-l-4
```

### グリッドレイアウト

```
ギャップ: gap-6
マージン上: mt-6
```

### リスト

```
行間: space-y-3
```

### 2カラム比較の濃淡ルール

- 左パネル: `bg-gray-50` + `border-gray-400`（明るい側）
- 右パネル: `bg-gray-100` + `border-gray-500`（暗い側）
- 微妙な濃淡で対比を表現する

## 文体ルール

- コロン（：）を使わない
  - NG: 「原則：自律性の最大化」
  - OK: 「原則 - 自律性の最大化」「原則❶ 自律性の最大化」
- 感嘆符（!）や疑問符（?）を避ける
  - NG: 「生産性が10倍に!」
  - OK: 「生産性が10倍に」
- 装飾的な絵文字を避ける
  - NG: 🎉🚀✨ など雰囲気だけの絵文字
  - OK: 📊🔄🎯 など意味を持つアイコン的使用は許容
- 色による強調は控えめに
  - NG: red-600, green-600 でテキストを色分け
  - OK: gray系の濃淡で強弱をつける
